# Changelog

All notable changes to the OIPS Reproducibility Package are documented here.
The project follows Semantic Versioning once the first public tag is created.

## [1.0.0] - Unreleased

Planned tag: `v1.0.0-manuscript`.

### Added

- Deterministic reconstruction of 733 cluster-v2 candidates from the reviewed
  1,742-record, 21-target public snapshot.
- Missing-aware static OIPS-P scoring and deterministic ranking.
- Clearly separated post-hoc reference, molecular-dynamics, and redocking
  evidence mappings.
- Frozen reference tables, source data, vector figures, and validation reports.
- Data manifests, checksums, manual-decision audit, asset-rights review, and
  external-archive inventory.
- One-command reproduction and verification CLI.
- Unit, integration, scientific snapshot, and release-security checks.
- Ten one-at-a-time ±20% static-module weight perturbations with scenario- and
  target-level stability outputs.
- Native five-tool and OIPS-P comparison on the 14-target complete-case subset.
- Article figure/table-to-output traceability index and converted-archive
  content comparison.
- English and Chinese documentation, equations, data dictionary, provenance,
  limitations, citation metadata, and dual licensing notices.

### Release gates

- Public repository URL, archival code/data DOIs, manuscript DOI, optional
  ORCIDs, and exact third-party versions/access dates remain pending.
- This version must not be tagged or archived until every blocking item in
  `PRE_PUBLICATION_CHECKLIST.md` is complete.
