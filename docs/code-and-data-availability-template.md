# Code and Data Availability — manuscript template

The public code and processed-data reproducibility repository for this study
will be available at **REPOSITORY_URL_PENDING**, release
**RELEASE_TAG_PENDING**, commit **COMMIT_SHA_PENDING**. It contains the source
code, fixed analysis configuration, processed pocket-feature inputs, frozen
reference outputs, statistical analyses, automated tests, repository-summary
figure source data, provenance records, an unresolved-case register, the ten
one-at-a-time ±20% module-weight perturbations, and the 14-system native
single-tool complete-case comparison. `MANUSCRIPT_RESULT_TRACEABILITY.tsv`
maps each article figure and table to its released source tables and identifies
the molecular artwork or trajectory-level panels that are not regenerated.
The code archive DOI is **CODE_DOI_PENDING** and the processed-data archive DOI
is **DATA_DOI_PENDING**; these identifiers must be replaced only after their
landing pages have been verified.

The repository does not contain complete raw molecular-dynamics trajectories,
restart/checkpoint files, commercial-software project files, licensed
binaries/logs, or unrestricted mirrors of third-party service outputs. The
provenance, redistribution boundary, and availability condition for included
and excluded materials are documented in `THIRD_PARTY.md`,
`data/metadata/asset-rights.tsv`, and
`data/external_archive_manifest.tsv`.
